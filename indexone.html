What is the Document Object Model (DOM)?

he Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a webpage as a tree of objects, where each node corresponds to a part of the document (like elements, attributes, and text). Through the DOM, programming languages like JavaScript can access and manipulate the content, structure, and styles of a webpage.

How do you select on element from the DOM?

Using getElementById, Using getElementsByClassName, Using getElementsByTagName etc.

What is the event delegation in the context of DOM,and why is it useful?

Event delegation is a pattern in JavaScript that allows you to handle events at a higher level in the DOM tree, rather than attaching an event listener to each individual element. Instead, you attach a single event listener to a parent element, and then use the event object’s target property to determine which child element triggered the event. This approach leverages the event bubbling phase, where an event propagates up the DOM tree from the target element to its ancestors. By attaching an event listener to a parent element, you can capture these bubbled events and respond to them accordingly.
